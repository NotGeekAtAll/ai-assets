# 介绍

`Kandy-pc`是一个轻量的移动端组件库，它基于信科的移动端规范，抽离了公用的组件，让页面更加统一。

## 技术栈

`Kandy-pc`基于以下技术栈开发

- [vue3](https://cn.vuejs.org/)
- [element-plus](https://vant-contrib.gitee.io/vant/#/zh-CN/home)
- [typescript](https://www.typescriptlang.org/)

## 浏览器支持

支持现代浏览器以及 Chrome >= 51、iOS >= 10.0（与 Vue 3 一致）
