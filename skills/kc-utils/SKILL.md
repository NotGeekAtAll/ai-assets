# KC Utils 方法库

在本仓库中开发、接入或复用 `@kc-one/kandy-utils` 时，使用这个 skill。

## 使用方式

1. 先阅读 `references/intro.md`，了解库的定位、技术栈和使用背景。
2. 在接入或修改代码前，先阅读 `references/quickstart.md`。
3. 编写具体组件或方法代码前，打开对应的 `references/<name>.md`。
4. 优先参考文档中的示例代码和 API 说明。
